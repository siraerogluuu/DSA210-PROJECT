# DSA210-PROJECT

The rapid growth of TikTok has made it one of the most popular social media platforms that millions of videos are shared every second in a day. To understand what makes some videos more successful than others has become an interesting question,mostly for content creators and marketers. That is the reason why I chose to focus on TikTok video popularity with a particular interest in whether posting time plays a role in performance in popularity. Moreover, I planned to use a dataset from Kaggle which contains informations about TikTok videos. This dataset includes variables such as view counts, like numbers, comments, duration of videos and usage of hashtags. Furthermore, it is expected to contain several thousand observations, which makes it suitable for my analysis. In addition to this dataset, I will add more data by including temporal feature derived from the posting time. These include the hour of the day, day of the week, and whether the video was posted on a weekday or weekend. These variables will help to understand how timing can influence engagement. Overall, this projects aim is to explore the patterns of this app within the dataset and provide insights into how different factors, mostly posting time relate to the video popularity.
